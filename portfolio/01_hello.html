Project Hello Quad

3D Graphics

Project Overview

Made a Quad that contains animations of vertex and fragment shader. It is to show the knowing about using OpenGL of makeing index and vertex buffer, vertex array, and texture. It also shows the ability of using vertex and fragment shader.

Tasks Completed

  • IndexBuffer-create indeces buffer, make the destructor of it, make use fuction to bind or unbind the index buffer.

  • Texture-Made LoadFromMemory to load from the memory to rander the image and maps it. Made UseForSlot to activate texture unit to the location where I want, made SetFiltering for fintltering. Made SetWrapping to wrap related to the S and T wrap. Lastly, made delete texture.

  • VertexArray-With constructor, I handled the vertex array and with the destructor, I deleted it. With use fuction, I can bind or unbind the vertex array. In SetIndexBuffer, I can get the index buffer and move the index to the index buffer. And in DrawIndexed and DrawVertices, I can draw Elements of index or arrays.(for each)

  • VertexBuffer-I set the buffer data to the buffer handle in the constructor and free it in the destructor. With use fuction, I can bind or unbind the vertex buffer. With BufferSubData in send_buffer_data_to_gpu, I can send the data.

Reflection

At first, I was confused about how to use OpenGL functions like GL_ARY_BUFFER, GL_Element_ARY_BUFFER, etc. So I checked docx.gl , found out the differences, heard more explanations about those functions, and finally made this project. I learned the lesson that it's very important to read documents correctly and how to use RenderDoc to check programs created by OpenGL.
